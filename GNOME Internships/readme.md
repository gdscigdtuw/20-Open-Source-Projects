## GNOME Internships &nbsp; <img src="https://static.gnome.org/img/gnome-logo.svg" width="60px">

- The program helps people from groups underrepresented in free and open source software get involved. 
- It provides a supportive community for beginning to contribute any time throughout the year and offers focused internship opportunities twice a year with a number of free software organizations.
- GNOME has participated in Google Summer of Code since its first year in 2005.
- GNOME has started and participated in Outreachy since 2010. 
- Visit https://wiki.gnome.org/Outreach
